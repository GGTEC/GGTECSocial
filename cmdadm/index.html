Comandos para moderadores

Esses são os comandos utilizados por moderadores no chat das lives, as respostas são exibidas na tela.

Comando Descrição Custo
!baluser Nome de usuário | Verifica o saldo do usuário. -
!baltake Nome de usuário 100 | Tira os pontos do usuário. -
!balgive Nome de usuário 100 | Dá pontos ao usuário. -
!fila Nome de usuário | Se o usuário tiver os pontos, ele é adicionado à fila de próximo. -
!rfila Nome de usuário | Remove o usuário da fila. -
!sr Nome da música | Comando provisório para moderadores, analisando como adicionar pedidos de música sem prejudicar as transmissões. -
!skip Pula a música atual. -
!volume VALOR Ajusta o volume de acordo com o valor digitado. Se não digitar nada, exibe somente o volume atual na tela. -
!esorteio Executa o sorteio e exibe o ganhador na tela, se houver algum sorteio no momento. -
!lsorteio Limpa a lista de participantes do sorteio. -
!csorteio Nome de usuário Verifica o nome de usuário na lista de sorteio. -
!addsorteio Nome de usuário Adiciona o nome de usuário no sorteio. -
